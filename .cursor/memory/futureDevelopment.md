# 今後の開発（開発者・エージェント向けメモ）

実装方針や検討事項を開発者・AI エージェント向けにまとめたメモ。仕様確定前の可能性も含む。

---

## AI・プロンプト

- **LLM の変更**: ChatGPT（OpenAI）から Gemini に変更する可能性あり。プレイリストタイトル・検索クエリ生成まわりを差し替える想定。
- **プロンプト**: プレイリストタイトル生成用プロンプトに手を加える可能性あり。

---

## 選曲・プレイリスト（Spotify 連携）

- **選曲フロー**: エージェント（AI）が Mood（天気・時間帯・ジャンルなど）から **Spotify Recommendations API** のパラメータを生成し、選曲する。
- **プレイリスト作成**: **Spotify Create Playlist API** でプレイリストを作成する。
- **ジャケ写**: 作成したプレイリストの先頭トラック（またはプレイリスト）のジャケ写を取得する。
- **UI**: 画面下部のジャケ写画像 UI と、レコードの中心にそのジャケ写を表示する。
- **遷移**: その UI から、Spotify の当該プレイリストページへ遷移させる。**このアプリ内では音楽の再生は行わない。**

---

## API 制限・代替案

- Spotify の API 制限（Development Mode の制限強化等）により、一時的に **Apple Music** または **YouTube Music** を代替として開発する可能性あり。

---

## モーダル

以下のモーダルを開発する予定。

- **初回アクセス時チュートリアル** — 初回訪問ユーザー向けのオンボーディング。
- **Spotify ログイン** — 未ログイン時にログインを促すモーダル。
